# 5 Bölüm Planı

## Genel
- **Bölüm 1–2:** Boşluğa düşme = dünya sınırında durur (güvenli).
- **Bölüm 3–4–5:** Boşluğa düşme = ölüm → aynı bölüm baştan.
- Hedef bölgesine girince: 1→2, 2→3, 3→4, 4→5. Bölüm 5 hedefi = oyunu bitir (level 1’e dön veya “Tebrikler”).

## Bölüm 1
- Mevcut harita (35×12). Zemin sürekli, zıplama + eğilme platformları.
- Hedef sağda → Bölüm 2.

## Bölüm 2
- Mevcut harita (45×14). Parçalı zemin, dar platformlar, eğilme geçitleri.
- Hedef sağda → Bölüm 3.

## Bölüm 3
- Daha zor: daha fazla boşluk, dar platformlar. **Düşme = ölüm** (y > harita altı + margin → bölüm 3 yeniden).
- Hedef sağda → Bölüm 4.

## Bölüm 4
- Daha zor: uzun boşluklar, zamanlı zıplama, mask değiştirme zorunlu.
- Düşme = ölüm.
- Hedef sağda → Bölüm 5.

## Bölüm 5
- En zor: çok dar platformlar, uzun boşluklar, birden fazla eğilme + mask.
- Düşme = ölüm.
- Hedef sağda = bitti → Level 1’e dön (veya “Tebrikler” ekranı).
